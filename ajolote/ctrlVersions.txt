----VERSION 0.02----

Version 0.02.01 --- 04/12/2021
    - Clases para margins y paddings
        * Cambio de clases del formato 'margen-izquierdo-2' a 'ml-2'
        * Las iniciales 'ml-2' vienen del inglés 'marginleft-2'
        * Adición de clases para margenes o paddings con la nomenclatura 'mv-2' o 'ph-2', donde 'm' es margin, 'p' es padding, 'v' es para vertical, 'h' para horizontal y el número corresponde al tamaño a aplicar
    - Clases para colores
        * Estandarizacion para clases que especifiquen el color del , ahora se usa la nomenclatura 'bg-nombreDelColor', además cada clase especifica un color de hover cuando la clase es puesta en conjunto con un btn
        * Estandarizacion para clases que especifiquen el color del texto, ahora se usa la nomenclatura 'clr-nombreDelColor'
